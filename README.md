# Tunes Alive: West Midlands
GPS-driven song discovery and uplifting the sense of community within the West Midlands through music sharing.

### <ins>Notes</ins>
- A portion of the entire mobile application project will be developed in response to the requirement of the university undergraduate module Mobile and Wearable Application Development (CMP6213).
